## Project Overview

This project is a responsive personal portfolio website created for the CSC4035 Web Programming and Technologies course. The goal of the assignment was to design and build a professional portfolio using **HTML5 and CSS3 only**, without using any frameworks like Bootstrap or Tailwind.

The website introduces who I am, the projects I have worked on, and provides a way for people to contact me. I designed the portfolio with a **developer-focused theme**, using a dark interface, monospace fonts, and a layout inspired by tools developers use such as GitHub and terminal interfaces.

The site is built using **modern web development techniques** such as Flexbox, CSS Grid, custom CSS variables, and responsive design.



## Website Sections

### Home / Hero Section

The hero section is the first section users see when they open the website. It introduces my name and includes a short developer-themed layout with call-to-action buttons and a simple statistics row that highlights my experience and projects.

### About Section

This section includes a short professional biography describing my background and interests as a Computer Science student. It also includes a profile image and highlights my main interests such as **Artificial Intelligence, web development, and software engineering**.

### Projects Section

The projects section showcases several example projects in card format. Each project card includes:

* Project title
* Short description
* Technologies used
* Image preview
* Links to GitHub or a live demo

The cards are designed using **CSS Grid and Flexbox** so they adjust nicely on different screen sizes.

### Contact Section

The contact section provides a simple form that allows users to send a message. The form includes fields for name, email, subject, and message. Basic HTML validation is used to make sure required information is filled in before submission.



## Technologies Used

The project was built using the following technologies:

## CSS Techniques Used

Some of the main CSS techniques used in this project include:

* **CSS Custom Properties (Variables)** for colors, spacing, and layout consistency
* **Flexbox** for navigation layout, project card elements, and the contact section
* **CSS Grid** for structuring larger page sections
* **Media Queries** for responsive design
* **CSS Animations and transitions** to add subtle visual effects



## Responsive Design

The website was designed using a **mobile-first approach** and adapts to different screen sizes.

Breakpoints used:

* **Mobile:** default styles
* **Tablet:** 768px and above
* **Desktop:** 1024px and above
* **Large screens:** 1200px and above

Screenshots showing how the site looks on different screen sizes are included in the **screenshots folder**.

## Challenges Faced

One challenge was making sure the layout looked good on both very small screens and large screens. This required adjusting font sizes, spacing, and layouts using responsive breakpoints.

Another challenge was creating a responsive navigation menu without relying on JavaScript frameworks. This was solved using a **CSS-only hamburger menu technique** that toggles the navigation links on smaller screens.
